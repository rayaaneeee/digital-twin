# AI Rayane — Personal Digital Twin

RAG-based chatbot that lets recruiters and visitors chat with a virtual version of Rayane Toumi on a portfolio site.

## Architecture

```
corpus/ (markdown files)
    ↓
ingest.py  →  Chroma vector DB
                   ↓
user message → retrieve.py (bge-small embeddings)
                   ↓
              persona.py (system prompt + context)
                   ↓
              Claude haiku-4-5 (streaming)
                   ↓
           FastAPI /chat endpoint (SSE)
                   ↓
           frontend chat widget
```

## Project structure

```
digital-twin/
├── corpus/
│   ├── cv.md                          ← your CV (already populated)

│   │   ├── dermai_medical_xai.md      ← DermAI project (already populated)

│   ├── retrieve.py     ← RAG retrieval
│   ├── persona.py      ← system prompt builder
│   ├── main.py         ← FastAPI server
│   ├── github_sync.py  ← auto-pull GitHub repos into corpus
│   └── requirements.txt
└── frontend/
    ├── index.html
    └── static/
        ├── css/style.css
        └── js/chat.js
```

## Setup

### 1. Install dependencies

```bash
cd digital-twin/backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Set your API key

```bash
cp .env.example .env
# edit .env and add your ANTHROPIC_API_KEY
export ANTHROPIC_API_KEY=sk-ant-...
```

### 3. (Optional) Sync GitHub repos into corpus

```bash
python github_sync.py
```

### 4. Add writing style samples

Create files in `corpus/style_samples/`:
- `style_sample_1.md` — a LinkedIn post, email, or README paragraph you wrote
- `style_sample_2.md`, etc.

These tune the AI's tone. 3–5 samples is enough.

### 5. Build the vector index

```bash
python ingest.py
```

Re-run this whenever you update corpus files.

### 6. Start the backend

```bash
uvicorn main:app --reload --port 8000
```

### 7. Open the frontend

```bash
# From the frontend/ directory
python -m http.server 3000
# → http://localhost:3000
```

Or just open `frontend/index.html` directly in a browser.

## Updating your data

| What changed | What to run |
|---|---|
| Updated CV or added a project | Edit `corpus/` files → `python ingest.py` |
| New GitHub repos | `python github_sync.py` → `python ingest.py` |
| New writing samples | Add to `corpus/style_samples/` → `python ingest.py` |

## Deployment

**Backend** → Render, Railway, or Fly.io free tier:
- Set env var `ANTHROPIC_API_KEY`
- Start command: `uvicorn main:app --host 0.0.0.0 --port $PORT`
- Upload `chroma_db/` with the code (or run `ingest.py` at build time)

**Frontend** → Vercel or Netlify:
- Update `BACKEND_URL` in `frontend/static/js/chat.js` to your deployed backend URL
- Deploy the `frontend/` folder as a static site

## Rate limiting

Default: 20 requests / hour per IP. Adjust `RATE_LIMIT` and `RATE_WINDOW` in `main.py`.
