# temacina-project
This project, developed in collaboration with Temacina, focuses on building an end-to-end data processing pipeline. It involves extracting text from documents using OCR, applying NLP techniques to analyze and structure the data, storing the processed information in a database, and presenting insights through an interactive dashboard.
